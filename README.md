# IPFS-Signal

## Goals

Bridge web2 to web3

- Push code to IPFS
  - Keep track of last few version

- Serve content over HTTP
  - Find owner based on domain
  - Find location on IPFS node
  - Send `x-ipfs-path`: record to user
  - Serve content

- Update DNS records to DNSLink ([https://dnslink.io/](https://dnslink.io/))
